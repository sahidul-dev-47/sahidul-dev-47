### 🏟️ SportVerse — Sports Facility Booking Platform
> A full-stack platform where venue owners list their sports facilities and athletes book time slots — no phone calls, no middlemen.

**Key Features:** JWT-secured REST API · Google OAuth · 4-step animated booking form · Owner dashboard (add/edit/delete venues) · User booking history with cancel support · Dark glassmorphism UI

**Tech:** `Next.js 15` `Tailwind CSS` `Framer Motion` `Express.js` `MongoDB` `Better Auth` `JWT`

### 🌍 Wanderlust — Travel Booking Web App
> A full-stack travel platform where users explore destinations, check prices with live tax breakdowns, and manage bookings from a personal dashboard.

**Key Features:** Browse & filter destinations by category · Live price calculator (base + 8% tax) · Date picker booking flow · Admin controls (add/edit/delete destinations) · Google OAuth + session auth · Fully responsive

**Tech:** `Next.js 15` `Tailwind CSS v4` `HeroUI` `Express.js` `MongoDB` `Better Auth`
